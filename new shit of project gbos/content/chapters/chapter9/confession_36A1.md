# SURFACE
看到了！我看到那张照片了！\n\n我不知道你是从哪找出这鬼东西的，妈的，这照片一定是假的，它不可能存在！\n\n天哪，可是照片里的最脏最丑的老东西就是我，对面那个……一定是艾莉丝。是她，没错。可是该死的，难道因为我脑子被波本威士忌烧穿了吗？为什么我脑子里根本没这一幕，哪怕一秒钟都没有！\n\n这到底是什么时候的事？我怎么可能完全不记得。

# DEEP
你能告诉我她给我看的是什么吗？还是说我知道那是什么，只是根本不敢承认……\n\n哈哈，我干嘛还要骗自己，这些年我跟艾莉丝相互寻找彼此，我们的目标始终是明确的，她要找到父亲，我则想在她找到瓦妮莎前把她干掉，我干嘛还要骗自己？？\n\n其实我早就知道了，早就知道瓦妮莎已经在她手里了，是吗？然后她去了埃尔帕索……操，她去那里干什么？？你快告诉我，她他妈的去埃尔帕索干什么？！

# CORE
天哪！我这该死的臭嘴到底吐了多少东西？我算什么？一个缩在酒瓶后发抖的寄生虫，被人像条狗一样牵着鼻子玩死。艾莉丝那时候一定在笑，看着我这个蠢货像死狗一样求她，然后哭着把一切都交代了。\n\n我不该活在这世上，求你了，扣扳机吧。照准了打，多开几枪，送我下地狱，现在就去。
