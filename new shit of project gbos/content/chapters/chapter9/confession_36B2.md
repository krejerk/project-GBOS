# SURFACE
我猜到了，你也觉得这件事不对劲。\n\n回想一下，你第一次听到青豆牡蛎汤计划这个词，是在什么时候？或许是某个人告诉你，说当我刚刚开始潜伏任务时，家族正在执行一个叫做这个名字的计划。至于它是什么意思，没人搞得清楚。\n\n老实告诉你吧，我第一次听到这个说法，就是在路易斯维尔，那天晚上我第一次见到父亲、瓦妮莎和其他人。几个小时后，我就端着一碗肉汤，和所有人围坐在一起。我还能记起来，那个瞬间，迷幻药的劲还没下去，隐约间我仿佛听到父亲嘴里冒出来这个词，青豆牡蛎汤计划，是什么呢？不知道，或许暂时不用在意。那时我正陶醉在一种奇怪的被包裹的温暖感里，那是迷幻药首次在身体里发挥作用的感觉。\n\n但在在那之后，我们为这个狗屁计划干过哪怕一件事吗？一件都没有。

# DEEP
但倘若细细回想，上车之后，很多 I 遭遇的事情至今都缺乏一个解释。比如亚瑟·道森死的时候，他的笔记本为什么会那么凑巧地塞在车底？那根本不是他临死前拼命藏起来的遗物，那是它们故意扔在那儿，专门等着我去捡的狗骨头！还有那两个被我亲手干掉的FBI探员……他们是怎么盯上我们的？结合你后来的发现，这只能是因为瓦妮莎的行踪早就被卖了。可目的又是什么呢？除此之外，最大的疑点在于，当年艾莉丝明明已经起疑心了。可后来呢？突然就没动静了。她闭嘴了，其他人也继续跟我喝酒、扯淡，就好像那件事根本没发生过。\n\n为什么？因为掩护得好？别扯了。一定是因为有人让艾莉丝把嘴闭上。在他们眼里，艾莉丝的“发现”根本算不上什么新闻。他们早就知道了。从我踏进那个房子的第一天起，我脑门上就贴着“警察”两个字，也许早在我在缅因州监狱的时候。回想起来，瓦妮莎看我的眼神，似乎更像是快淹死的人看见救生圈的眼神。她也知道我是条子，所以才认定，我能把她从那个泥潭里捞出去。

# CORE
可她知道，父亲会不知道吗？他眼睁睁地看着瓦妮莎往我这边靠，看着她在心里偷偷打包行李。他把一切都看得清清楚楚。我自以为聪明，拿着瓦妮莎当诱饵去设伏，谁会想到父亲也正在拿瓦妮莎当诱饵。他把这块带着血的肉挂在钩子上，就坐在高处，看着我这头自作聪明的看门狗闻着味儿、一步步踩进他早就下好的夹子里，他知道只要瓦妮莎一天不安全，我就不会威胁到他的生命安全。\n\n所以，这所谓的“青豆牡蛎汤计划”到底是什么呢？我想，只有一种解释。它是在得知我要上车之前，天才的父亲想到的一个绝妙计划，给我，罗伯特·卡彭专门准备的计划。它的真正目的，就是要借我的存在，借灰水信标的传话系统，反向渗透进FBI情报网络，实现一种逆向操控，并且在未来的某个时候，还能当成一张牌打出去。\n\n而且你再想想看，既然他们知道我的底细，会不会顺水推舟，早早安插个人进系统？\n\n詹妮弗在所有问题上的倾向性，她难道不是另一张底牌吗？藏在该死的机器后边，冷眼看着我在泥潭里翻滚，看我为了一堆白骨痛哭流涕，她一定觉得，我这条老狗永远翻不出她的手掌心。\n\n我这辈子被他们毁得一干二净，但我现在醒了。感谢上帝，他们是如此的想要父亲活下去，而我却恰恰握有那把掌控生杀大权的钥匙。\n\n听着，你曾经给我看过一封索恩探员写给雷吉博士的信，还记得吗？我要你找到那封信，把那串被涂抹掉的代码和他的设施识别码复制给我。
